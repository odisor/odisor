## üí° Featured Projects

### [Azure VMs OS Metrics Alert Deployment Pipeline](https://github.com/odisor/poc-azure-ama-monitoring)

An **idempotent Infrastructure-as-Code deployment pipeline** demonstrating production-ready monitoring automation for Azure Virtual Machines. This proof-of-concept leverages ARM templates to provision Azure Monitor Agent (AMA) with OS-level metrics collection and intelligent alerting rules.

**Key Features:**
- Fully parameterized and dynamic pipeline architecture
- Configuration-driven alert management via external files
- Reusable templates for rapid deployment across environments

<br>

### [Azure Virtual Networks Connectivity Testing](https://github.com/odisor/poc-azure-virtual-networks-testing)

A comprehensive **network validation sandbox** for Azure Virtual Network configurations. This project provides a flexible testing framework for validating VNet topologies, connectivity scenarios, and routing behaviors.

**Use Cases:**
- VNet peering and subnet connectivity validation
- Network security group (NSG) rule verification
- Routing table testing and troubleshooting
- Isolated environment connectivity scenarios
